# **App Name**: Axiom EdTech

## Core Features:

- Dynamic Landing Page Display: Renders a modern, section-based landing page including Hero, Features, Courses Preview, Why Choose Us, Testimonials, Pricing, CTA, and Footer sections, ensuring full responsiveness for all devices.
- Interactive AI Learning Assistant: A floating chatbot widget that provides context-aware answers to user queries about the company, courses, and pricing using a preloaded mock JSON data knowledge base to simulate GPT-like behavior.
- Unified Light and Dark Mode Toggle: Allows users to switch between a clean light and an 'ultra-modern SaaS aesthetic' dark theme for the entire application, maintaining visual consistency.
- Enhanced UI Animations & Effects: Implements Framer Motion for smooth scroll effects, subtle animated background particles, floating blurred gradient blobs, glassmorphism card hover effects, and gradient border animations to create a dynamic and premium user experience.
- Structured Course & Pricing Presentation: Displays course cards with images, instructors, ratings, and 'Enroll Now' buttons, supporting horizontal scroll on mobile and a clean grid on desktop. Also includes a pricing section with 3 tiers and an animated monthly/yearly toggle.

## Style Guidelines:

- Primary interactive color: A vibrant electric indigo (#5E76ED), serving as the main hue for CTAs, highlights, and primary interactive elements.
- Background color: A deep, subtly blue-tinted dark gray (#16171D), providing a rich contrast for the modern SaaS aesthetic.
- Accent color: A striking gold (#F5C73D), used sparingly to provide subtle highlights and emphasize key visual components.
- Headline and body font: 'Inter' (sans-serif) for its modern, neutral, and highly readable characteristics, suitable for a professional SaaS platform. Note: currently only Google Fonts are supported.
- Utilize modern, crisp, and minimal icons, consistent with the ShadCN UI aesthetic, for features, social media, and chatbot avatar.
- Adhere to a mobile-first responsive design approach with clean spacing, employing a mix of grid and comparison layouts where appropriate, and specific horizontal scrolling for course cards on smaller screens.
- Incorporate smooth transitions, hover effects, subtle background motion, and targeted UI animations (e.g., pricing toggle, chatbot modal) powered by Framer Motion to enhance user engagement without distraction.
